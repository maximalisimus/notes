Mikhail Artamonov - автор предоставленного вебсайта, содержащего статичные страницы с информацией определённого характера.
Автор имеет широкие познания как в области программирования, так и в области развлечений.

Дисклеймер.

Данная информация представляет из себя субъективные заметки, которые определяют вкусы лишь самого автора и несёт лишь ознакомительный характер.
Автор не несёт никакой ответственности за предоставленную информацию. Некоторые из списков запрещены к просмотру лицам до 18+ лет.

При помощи данных заметок осуществляется быстрый доступ к информации о просмотренных фильмах, которые сильно понравились автору. Также осуществляется быстрый доступ к списку прочтенных книг.
Т.к. вебсайт находится в репозитории Github (с включенной функцией хостинга GItHub Pages), то при помощи системы контроля версий Git легко осуществляется управление, а также восстановление данных в случае фатальной ошибки автора.
В случае работы вебсайта на любой из CMS, для восстановления данных при фатальной ошибке автора потребуется значительно больше времени, а в некоторых случаях может стать и вовсе невозможным.
Однако, основная причина создания данного веб-сайта именно на хостинге GItHub Pages в качестве статичного, совершенно иная.
При помощи данного веб-сайта, осуществляется накопление первоначальных навыков работы с языком JavaScript, а также совершенствование знаний в языке CSS.

На странице с книгами имеется отдельный предпоследний список, который содержит в себе информацию о книгах, которые хотелось бы прочитать автору в ближайшем будущем.

Данные заметки регулярно пополняются новыми списками просмотренных фильмов и сериалов, а также списками новых прочтеных книг.

На странице книги далеко не у всех книг имеется кнопка - "Подробнее". Однако, она всё-таки имеется.
После нажатия на указанную кнопку отображается подробная информация о книге. Данная информация содержит анотацию, возможность скачать книгу в одном из указазнных форматов, а также возможность читать книгу онлайн, не скачивая её к себе на ПК.
Страница с книгами будет постепенно пополняться подробной информацией с возможностью скачивания или чтения онлайн.
В данный момент подробная информация о каждой книге находится в разработке.

На странице со списком фильмов - полного режима, не компактного, также лишь у некоторых строк имеется кнопка "Анотация", аналогичная кнопке "Подробнее" на странице с книгами.
При нажатии на данную кнопку отображается подробная информация о выбранном фильме, содержащая в себе наиболее важные, по мнению автора, данные.
В данный момент подробная информация о каждом фильме находится в разработке.
